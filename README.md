## Installation
```
git clone git@github.com:dn16/react-spa-template.git
cd react-spa-template
rm -rf .git/
yarn install
```

## Script usage
| Command        | Description                                                   |
|----------------|---------------------------------------------------------------|
| start          | Starts production server; served at `localhost:8080`          |
| build          | Bundles the source in `~/public` directory                     |
